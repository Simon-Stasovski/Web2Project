(async) addCard(cardName, type, description, serialNumber, frontImagePath, backImagePath, isForSale, cardCondition, certificateImage, cardPrice, cardOwner)

Adds a new card entry to the card table with the passed in data. Uses the isValid function to validate the passed in data. It also converts the valid data into a certain styling format to ensure consistency (ex: all card names are lowercase). Returns the object representation of the added card. The database, connection and card table are initialized before the function is called.
